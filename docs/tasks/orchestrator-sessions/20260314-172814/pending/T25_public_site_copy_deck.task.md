## Objective

Write reusable, school-friendly marketing copy for the public website, including home-page messaging, value propositions, trust signals, and conversion CTAs.

## Scope

Included: primary page copy, headline system, CTA language, reusable school-variable placeholders.  
Excluded: SEO metadata implementation and coded pages.

## Context

This copy will inform `apps/www` and should work for the current school while remaining portable to future schools.

## Definition of Done

- Core page copy exists for the public site.
- Messaging supports admissions and general school trust-building.
- White-label placeholders are clear where school-specific details change.

## Expected Artifacts

- `docs/features/public-site-copy.md` or equivalent copy deck location

## Constraints

- Keep the voice credible, warm, and school-appropriate.
- Avoid generic SaaS jargon.

## Verification

- Confirm the copy deck covers home, about, academics, admissions/contact, and login entry points.
