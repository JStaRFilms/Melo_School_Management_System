# [FR-011] Public Website and Admissions-Ready Content

## Labels
`MUS`, `website`, `marketing`

## User Story
As a prospective family, I want a polished school website with clear information and calls to action, so that I can learn about the school and contact it easily.

## Proposed Solution

### Overview
Build a shared public-site engine with tenant theming, structured page content, and reusable blocks for school storytelling and conversion.

### Implementation Flow
1. Define public-site page model and tenant branding rules.
2. Build home and supporting public pages.
3. Add contact and admissions-ready CTA blocks.

### Key Considerations
- The design must feel premium and intentional, not generic.
- The site must stay reusable for later schools.

## Acceptance Criteria

- [ ] The public site supports school-specific branding and copy.
- [ ] Core marketing pages are implemented and mobile-first.
- [ ] Public CTAs and contact intent flows are present.
