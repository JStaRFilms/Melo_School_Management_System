# [FR-010] Teacher Workspace

## Labels
`MUS`, `teacher`, `workspace`

## User Story
As a teacher, I want a dashboard for lesson planning, class work, results, and assessment tools, so that my daily workflow lives in one place.

## Proposed Solution

### Overview
Create a dedicated teacher app with dashboard summaries, class assignment views, lesson-note planning, results workflows, and AI entry points.

### Implementation Flow
1. Build teacher navigation and dashboard modules.
2. Add assignment-aware class and subject screens.
3. Connect lesson planning and results entry flows.

### Key Considerations
- Teacher UX should prioritize speed on mobile devices.
- Access should be limited to assigned classes and subjects.

## Acceptance Criteria

- [ ] Teachers have a dedicated workspace with role-specific navigation.
- [ ] Teachers can access assigned classes and subjects quickly.
- [ ] Workspace exposes planning and result-entry flows.
